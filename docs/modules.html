GlyphKeysTemplate

GlyphKeysTemplate: [string, string]
  • Defined in widgets/VirtualKeyboard/KeyRow.ts:33

A template for multiple GlyphKey virtual keyboard keys. A 2-tuple of strings, where each string has the same length. Each character of the string represents a glyph to add to a keyboard row. The first string of the tuple has the regular glyphs, while the second string string of the tuple has the alternative glyphs.

Example:

const template: GlyphKeysTemplate = ['qwertyuiop', 'QWERTYUIOP'];

Helper Functions

measureTextDims

  • measureTextDims(text: string, font: string): [number, number, number]
    • Defined in helpers/measureTextDims.ts:18

    Measures the dimensions of a given string of text with a given font.

    Note that the first time calling this function is slower than subsequent calls because a dedicated canvas context must be created.

    Parameters

    • text: string
    • font: string

    Returns [number, number, number]

    Returns a tuple containing, in this order, the width of the text, the text's ascent length and the text's descent length.

    The actual height can be found by summing the ascent and descent.
