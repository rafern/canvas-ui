Class Viewport

Viewports are internally used to manage a canvas' size and painting. It is used by Root and ViewportWidget.

populateChildsLayout

  • populateChildsLayout(child: Widget): null | LayoutContext
    • Defined in core/Viewport.ts:86

    Populates the given child's layout by calling Widget.populateLayout.

    If forceLayout is true, then it is reset to false and Widget.forceLayoutDirty is called.

    If the child's layout is not dirty, then populateLayout is not called and no layout context is returned, else, a layout context is returned which should be used with resolveChildsLayout.

    Parameters

    • child: Widget

    Returns null | LayoutContext

resolveChildsLayout

  • resolveChildsLayout(child: Widget, layoutCtx: null | LayoutContext): boolean
    • Defined in core/Viewport.ts:120

    Resolves the given child's layout with a given layout context by calling Widget.resolveLayout.

    If the child's layout is not dirty or the given layout context is null, then resolveLayout is not called.

    Expands canvas if the new layout is too big for the current canvas. Expansion is done in powers of 2 to avoid issues with external 3D libraries.

    Parameters

    • child: Widget
    • layoutCtx: null | LayoutContext

    Returns boolean

    Returns true if the child was resized, else, false.
