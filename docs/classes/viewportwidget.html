Class ViewportWidget

A type of container widget which is allowed to be bigger or smaller than its child.

Allows setting the offset of the child, automatically clips it if neccessary. Otherwise acts like a Container. Implemented by using a Viewport; effectively, the child widget is painted to a dedicated canvas.
