Class ViewportWidget<W>

A type of container widget which is allowed to be bigger or smaller than its child.

Can be constrained to a specific type of children.

Allows setting the offset of the child, automatically clips it if neccessary. Otherwise acts like a Container. Implemented by force re-painting the child and clipping it or, optionally, by using a Viewport to paint the child widget to a dedicated canvas.
