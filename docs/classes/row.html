_children: W[]

Inherited from MultiContainer._children

  • Defined in widgets/Parent.ts:32

This widget's children. Note that this is marked as readonly so that it cannot be accidentally replaced with a new array. This way, references to this array are always valid. If you want to clear this array, set the length to zero instead of creating a new instance. readonly still means that you can add/remove elements to/from the array.

See children for the public iterator getter.
