Class VirtualKeyboard

A virtual keyboard widget.

Needs a KeyboardDriver so that key events can be queued.

Equivalent to creating a Column of VirtualKeyRow with a shared KeyContext. Key rows will be created with SpaceBetween main alignment and Stretch cross alignment.
