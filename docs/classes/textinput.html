Class TextInput<V>

A flexbox widget that allows for a single line of text input.

Supports obscuring the text with hideText, which shows all characters as black circles like in password fields, text validation and toggling editing.

If a TextInputHandler is set, then that will be used instead of keyboard input for mobile compatibility.

    The type of validValue; the type of the transformed value returned by the validator.
