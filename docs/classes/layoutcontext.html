Class LayoutContext

A layout context is an object used to keep track of how much flex ratio and basis each Widget wants when resolving layout. It also keeps track of the maximum width and height flexbox-like layouts can expand to and the verticality of the layout.

addBasis

  • addBasis(hBasis: number, vBasis: number): void
    • Defined in core/LayoutContext.ts:63

    Add basis to this layout context.

    If this is a vertical layout, vertical basis will be added to vBasis while horizontal basis will be set to the maximum of the new horizontal basis and hBasis.

    If this is not a vertical layout, horizontal basis will be added to hBasis while vertical basis will be set to the maximum of the new vertical basis and vBasis.

    • vBasis: number
